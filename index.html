Workshop Overview

This hands-on workshop introduces the NGS data pipeline for population genetics using Illumina platform data. Focus is on WGS resequencing, though adaptable for RAD-seq/amplicon-seq.

You will learn to:

  • Handle FASTQ files
  • Conduct quality control and trimming
  • Map reads to reference genomes
  • Call and filter SNPs/indels

Pre-requisites & Setup

Use a Unix/Linux shell for all operations. Windows users: install MobaXterm.
