Genomics Workshop

This hands-on workshop introduces students to the basic analysis pipeline for Next-Generation Sequencing (NGS) reads generated using Illumina platforms, with a focus on population genetics.

The workshop primarily uses whole genome resequencing (WGS) data, but the pipeline can also be adapted for RAD-seq or amplicon-seq data when a reference genome is available.

By the end of the workshop, participants will understand how to:

  • Work with raw sequencing data (FASTQ files)
  • Perform quality control and preprocessing
  • Map reads to a reference genome
  • Call and filter genetic variants (SNPs/indels)

Pre-requisites & Setup

There are two ways to run this workshop. One option is using Google Colab, which allows you to run the analysis without any local computational setup, as long as you have a Google account and Google Drive access.

Alternatively, you may run the workshop on your local machine using a Unix/Linux shell environment.

Windows users may install MobaXterm. Mac and Linux systems already include a terminal.
